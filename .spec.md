## What It Does
The finance-blog project is a static website focused on financial advice and education. The site includes articles about personal finance management, investment strategies, and budgeting tips.

## Tech Stack
- Language: JavaScript (for dynamic parts)
- Framework: Hugo (static site generator)
- Database: None (uses YAML files for configuration)
- Key libs: Google AdSense, Google Analytics

## Entry Points
- `index.html` — main landing page
- `.git/hooks/pre-commit` — script to audit blog before commit

## File Structure
```
/content/posts/ - contains all articles in markdown format
/assets/images/ - images used across the site
/config.yaml - global configuration settings
/templates/ - custom Hugo templates for layout
/static/js/blog-audit.js - script to check links and content consistency
```

## How to Run
`hugo server --theme=hugo-bootstrap-v5`
